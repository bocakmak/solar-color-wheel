# solar_color_wheel

The Circuit
======================================================================

Board
Arduino Nano (Atmega328)

Components

1 6.5 volt, 1.5 watt Solar Panel
1 3 AA battery pack
3 AA batteries

3 330ohm resistors
1 100 ohm resistor
1 4001 Diode

1 Common anode RGB LED

What it Does
======================================================================

Loops through an array of pre-defined colors, blending all colors 
between the previous and next RGB color set.
